# Product Overview

Pharma CI Platform is a comprehensive pharmaceutical competitive intelligence monitoring system. It provides real-time tracking and analysis of the pharmaceutical industry landscape.

## Core Features

- Dashboard with key industry metrics and visualizations
- Competitor Pipeline tracking across all development phases (Preclinical through Approved)
- News Monitoring with sentiment analysis
- KOL (Key Opinion Leader) Insights and publication tracking
- Real-time data visualization using Plotly charts

## Data Sources

The platform integrates data from:
- PostgreSQL database (primary data store)
- ClinicalTrials.gov (clinical trial data)
- PubMed (scientific publications)
- FDA Database (regulatory information)

## Target Users

Pharmaceutical industry professionals who need to monitor competitor activities, track drug development pipelines, and stay informed about industry trends and key opinion leaders.
